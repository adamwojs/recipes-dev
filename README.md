# Ibexa recipes repository

Here is the repository for `ibexa/*` and `ezsystems/*` packages. In order to use the recipes, configure your project with `flex.ibexa.co` server:
```json
# composer.json
        
{
    ...
    "extra": {
        "symfony": {
            "allow-contrib": true,
            "endpoint": "https://flex.ibexa.co"
        }
    }
}
```
